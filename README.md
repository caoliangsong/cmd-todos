# cmd-todos
a todo list in command line 

# how to use

```
cmd-todo get 
```
get all list

```
cmd-todo get done
```
get done list
```
cmd-todo get undone
```
get undone list

```
cmd-todo add xxx 
```
add xxx to  list

```
cmd-todo done xxid
```
set id=xx done

```
cmd-todo undone  xxid
```
set id=xx undone
