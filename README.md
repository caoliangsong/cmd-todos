# cmd-todo
a todo list in command line 

# Install
```
npm install cmd-todo -g
```

# How to use
cmd-todo -h
## Work at command line

### When first use, auto register new user

```
cmd-todo get 
```
get all list

```
cmd-todo get done
```
get done list
```
cmd-todo get undone
```
get undone list

```
cmd-todo add xxx 
```
add xxx to  list

```
cmd-todo done xxid
```
set id=xx done

```
cmd-todo undone  xxid
```
set id=xx undone
